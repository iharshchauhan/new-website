# Vibe Coded App 1

## What it does

This app is a showcase of how to build an application entirely based on vibes. It uses modern web technologies to deliver a seamless and interactive experience.

## Why built

I built this app to explore the boundaries of interactive design and to see how far I could push the concept of "vibe coding" — building software that feels good to use, not just functional.
