# Thought Process & Decisions

When building Vibe Coded App 1, I had to make several key decisions:

1. **Tech Stack**: Chose Next.js for its robust routing and server-side rendering capabilities.
2. **Styling**: Tailwind CSS was a no-brainer for rapid prototyping and maintaining a consistent design system.
3. **Animations**: Framer Motion provided the necessary tools to add that extra "vibe" to the interactions.

The main challenge was balancing performance with the heavy use of animations. I opted for CSS-based animations where possible and reserved Framer Motion for complex orchestrations.
